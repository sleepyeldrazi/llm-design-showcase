# Second Brain Landing Page - Design System

This project contains five distinct design iterations for a note-taking application/second brain concept.

## Design Overview

Each design iteration represents a different design philosophy:

### Design 1 (slash-one)
- **Style**: Minimalist Clean
- **Focus**: Simplicity and speed
- **Key Features**: Clean whitespace, fast performance, straightforward messaging

### Design 2 (slash-two)
- **Style**: Modern Dark Theme
- **Focus**: Professional and modern
- **Key Features**: Dark mode, tech-focused, AI-powered features highlighted

### Design 3 (slash-three)
- **Style**: Creative Visual
- **Focus**: Educational and inspiring
- **Key Features**: Bold typography, visual hierarchy, motivational messaging

### Design 4 (slash-four)
- **Style**: Professional Corporate
- **Focus**: Enterprise-grade features
- **Key Features**: Clean, professional, trustworthy design

### Design 5 (slash-five)
- **Style**: Modern Educational
- **Focus**: Growth and learning
- **Key Features**: Gradient backgrounds, user-focused messaging

## How to Use

1. **Main Entry Point**: Open `index.html` in a browser to see the main landing page with the design switcher.

2. **Switch Designs**: Use the buttons at the top of the page to switch between different design iterations.

3. **View Designs**: Each design is contained in its own subdirectory:
   - `slash-one/index.html`
   - `slash-two/index.html`
   - `slash-three/index.html`
   - `slash-four/index.html`
   - `slash-five/index.html`
