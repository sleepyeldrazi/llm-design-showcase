# Viewing Your Designs

## Quick Start

1. **Open the main page**: Open `index.html` in your web browser
2. **Browse designs**: Use the buttons at the top to switch between the five design iterations
3. **View individual designs**: Each design is contained within its own directory

## Customization

You can modify any of the designs by editing the HTML and CSS files:
- `index.html` - Main content structure
- `style.css` - Design-specific styles

## Features

- **Responsive**: Works on mobile and desktop
- **Easy Switching**: Click buttons to instantly view different designs
- **Customizable**: Easy to modify colors, fonts, and content
