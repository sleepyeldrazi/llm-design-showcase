# The Lab — Small Model Showcase

A side-by-side comparison portal for landing pages generated by different small language models, with and without frontend-design prompting.

## Structure

| Directory | Contents |
|-----------|----------|
| `portal/` | Vite + React portal app (the viewer) |
| `gemma/`, `gemma_frontend/` | Pages by Gemma 4 (26B) |
| `glm/`, `glm_frontend/` | Pages by GLM-4.7 Flash (30B) |
| `qwen/`, `qwen_frontend/` | Pages by Qwen 3.5 (35B) |
| `qwen27/`, `qwen27_frontend/` | Pages by Qwen3.6 27B |

Each model produced **5 design iterations** for the same prompt:

> *Design the landing page for a note-taking application as essentially a second brain. Create five iterations accessible via `/one`, `/two`, `/three`, etc., with a switcher to toggle between them.*

The `_frontend` variants were generated **with a frontend-design skill / system prompt**; the plain folders were generated raw.

## Prerequisites

- [Node.js](https://nodejs.org/) 18+

## Start locally

```bash
cd portal
npm install
npm run dev
```

The dev server starts on `http://localhost:43234` and exposes the portal UI. A custom Vite plugin also serves the sibling static sites under short path prefixes (e.g. `/g/`, `/gf/`, `/l/`, etc.) so everything works from a single origin.

### Build for production

```bash
cd portal
npm run build
npm run preview
```
